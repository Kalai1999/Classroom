# Backend 

Install Nestjs and npm

## Nest install Command

"npm i -g @nestjs/cli"

## Node and Npm Installer

https://nodejs.org/en/download/

## Open Project Folder 

run "npm i" to install all dependencies

## Run

npm run start:dev

# Frontend

Install React and its dependencies

## Open Project Folder 

run "npm i" to install all dependencies

## Run

"npm start"
